# Python Basics Collection

## 🚀 Projects Overview

Here is a breakdown of the scripts included in this repository:

* **Calculator (`Calculator.py`)**: A basic calculator application capable of handling core arithmetic operations.
* **Countdown Timer (`Countdown Timer.py`)**: A simple utility script to count down time.
* **Multiplication Table Generator (`Multiplication Table Generator.py`)**: Generates custom multiplication tables based on user input.
* **Simple Interest Calculator (`Simple Interest Calculator.py`)**: Calculates interest based on principal, rate, and time inputs.
* **Speeds using lists (`Speeds using lists.py`)**: Explores Python lists by calculating minimum, maximum, and average speed data.
* **Student Profile Generator (`Student Profile Genrator.py`)**: A script designed to format and display student information profiles.
* **Student Grade Calculator (`Student-Grade-Calculator.py`)**: Takes scores as inputs and automatically computes final grades.
* **Student Marks Manager (`student-marks-manager v1.1.py`)**: A tool to track, update, and manage student grades (latest version).
* **Age Calculator (`age-calculater.py`)**: Computes a user's exact age based on birth dates.
* **Hello World (`hello.py`)**: My very first Python script!
